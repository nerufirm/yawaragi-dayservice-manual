第1章 一日の業務フロー（日次業務）

デイサービスの一日の流れを、時間帯ごとに示します。各業務の詳細手順に従い、安全で快適なサービス提供を心がけてください。

出勤後〜8:35

1.1 朝の準備

環境整備

  • ポット内に水を補充する
  • カーテン・窓を開けて換気する
  • 洗面台の電源を入れる
  • トイレの座面等のスイッチをONにする
  • 和室ベッドの電源を入れ、布団を整える

浴室準備

  • 脱衣所と浴室内の準備を行う
  • 8:30頃にお湯張りを開始する

消毒・衛生

  • 各バケツに消毒液（オスバン・カラーブリーチ等）を準備する
  • 血圧計・体温計をセットする
8:35〜12:00

1.2 お迎え〜午前の業務

お迎え（8:35）

  • 利用者のお迎えに出発する
  • リーダーは施設に残り、バイタルチェック・再検を行う
  • 入浴用オレンジの袋を持ってくる
  • 曜日により酸素濃縮装置を持ってくる

デイ開始（8:55）

  • リーダーが挨拶を行う
  • ラジオ体操を実施する
  • 飲み終わったコップ類を片付けて洗う

バイタル測定

  • 来所後すぐにバイタルを測定する（体温・血圧・脈拍）
  • 異常値が出た場合は必ず再検する
注意: バイタル異常値の基準 — 血圧上180/下110以上、体温37.5℃以上、脈拍100以上/50以下の場合は再検し、リーダーに報告すること。

入浴介助（9:00頃〜）

  • 基本4名体制で入浴介助を行う
  • 利用者の自立度・身体的特徴に応じた個別対応を行う（第4章参照）

フロア活動

  • 入浴待ちの利用者に脳トレ（塗り絵、パズル等）を提供する
  • 音楽を流す
  • 薬の確認を行い、実施票に黒ペンでチェックする

昼食前準備（11:00〜）

  • テーブルを拭き、手指消毒を行う
  • トイレ誘導を実施する
  • テーブル席への誘導を行う

配膳・服薬（11:40〜11:45）

  • 食事を配膳する（食事形態を必ず確認すること）
  • 服薬を確認し、実施票にチェックする
12:10〜13:50

1.3 昼食〜午後の業務

食後対応（12:10頃〜）

  • 早く食べ終わった利用者から下膳する
  • 口腔ケアを実施する
  • トイレ誘導を行う
  • 口腔・トイレ終了後、昼休憩する場所へ誘導する
  • エプロンを洗濯し、たたむ

職員交代（12:30〜）

  • 遅食職員は落ち着いたら記録に入る
  • 早食の職員が戻ってきたら交代する
  • 昼休憩時の見守りを行う

午後の準備（13:25〜）

  • レクリエーション活動の準備をする
  • 離床、トイレ誘導を行い、席へ誘導する
13:50〜15:00

1.4 午後体操・レクリエーション

  • 窓を開放して空気の入れ替えをする
  • 体操を実施する（約20分）— 詳細は第5章参照
  • 口腔体操を行う（「パタカラ」「あいうべ」体操）
  • レクリエーション活動を実施する
15:00〜16:30

1.5 おやつ〜帰宅

おやつ提供（15:00）

  • おやつを提供する（利用者ごとの形態を確認すること — 第4章参照）
  • 飲み物を配る

終了体操（15:55）

  • 「ボケたらあかん体操」を行う

帰宅準備・送迎

  • 帰りの準備を行い、ホームへ誘導する
  • 送迎を実施する
  • 施設の片付け・消毒を行う

第2章 生活相談員業務（スケジュール管理・事務）

2.1 週次業務（毎週金曜日）

毎週金曜日に、翌週分の以下の書類を作成・配布する。

書類名配布先備考
利用者週間スケジュール表デイ・厨房・ホーム翌週の利用者予定
バイタル記録表デイ測定記録用
週間入浴種別一覧表デイ一般浴・リフト浴等の区分

2.2 月次業務

時期業務内容詳細
12日〜19日希望休受付職員の希望休を受け付ける（2日間まで優先）
25日前後勤務表作成・配布翌月の勤務表を作成し全職員に配布
20日以降カンファレンス実施社長、ホーム主任が参加。利用者の状態・ケア方針を共有

2.3 月末処理

  • 翌月の週間プラン・月間プランを作成する
  • 月初にプランを保存する
  • 利用日変更に伴う食せんを厨房へ提出する
  • 担当ケアマネジャーへ以下の書類を配布する:
    • サービス提供状況報告書
    • 計画書
    • モニタリング

第3章 記録業務・システム入力

3.1 デイリー入力（タブレット操作）

毎日の業務終了後、タブレットから以下の項目を入力・保存する。

入力項目内容注意事項
出欠出席/欠席/中止理由も記載する
バイタル体温・血圧・脈拍異常値は特記事項にも記載
入浴一般浴/リフト浴/非実施非実施の場合は理由を記載
排泄排尿・排便の回数・状態パット確認の結果も記載
食事主食・副食の摂取量形態（普通・刻み・ミキサー・粥）を確認
レク実施状況と特記（様子）参加状況・表情なども記録

3.2 AppSheet入力（ホーム連携）

ホームとの情報共有のため、AppSheetアプリで以下を入力する。

項目入力方法
バイタル通常入力
入浴通常入力
食事追加記録で対応
薬通常入力
排便新規記録で入力
排尿追加記録で対応
ポイント: 排便は「新規記録」、排尿・食事は「追加記録」で入力する点に注意してください。

第4章 個別ケア・食事・おやつ

4.1 食事・おやつ形態の管理

利用者ごとに食事・おやつの提供形態が異なります。必ず「おやつ形態一覧表」を確認してから提供してください。

形態説明
普通通常の形態でそのまま提供
刻み細かく刻んで提供（咀嚼機能低下の方）
ミキサーミキサーにかけてペースト状に（嚥下機能低下の方）
粥主食を粥に変更して提供

4.2 入浴介助の個別対応

利用者の身体的特徴・既往歴を把握し、安全な入浴介助を行ってください。

注意項目対応
左目視力なし左側からの声かけ・誘導に配慮。浴室内での左側からの接近に注意
心臓疾患あり湯温・入浴時間に注意。急な温度変化を避ける
手足の冷え事前に足浴等で温める。湯温を確認してから入浴
難聴大きな声でゆっくり声かけ。ジェスチャーを併用
リュウマチ関節への負担に注意。ゆっくりとした介助
逆流性食道炎食後すぐの入浴を避ける
携帯酸素使用入浴時は在宅酸素の機械に切り替える
パット使用パットの確認（濡れていることがある）。交換を行う

洗体の順番

利用者の状態に応じて、以下の部位を丁寧に洗う:

  • 胸 → 背中 → 前側 → 下肢（指の趾間）→ 陰部

4.3 イベント・外出レクの注意点

外出レク（ドライブ等）の注意事項（過去の改善点を反映）:
  • 乗り物酔い対策として、酔い止め薬を事前に準備する
  • 出発・到着の時間を厳守する
  • 水分補給用の飲料を事前に準備する
  • トイレの場所を事前に確認しておく

カラオケレクで使用する曲目の例

  • 北海盆唄
  • 千曲川
  • その他利用者のリクエスト曲

第5章 体操・口腔ケア・レクリエーション

午後の体操は13:50〜14:10に実施します。以下のプログラムを順番に行ってください。

5.1 午後体操プログラム（上半身）

順番声かけ（ひらがな）内容（スタッフ用）回数
1しんこきゅうをします深呼吸を3回します。3回
2くびをまえとうしろにうごかします首を前（おへそを見る）後ろ（天井を見る）と動かします。3回ずつ
3くびをみぎひだりにうごかします首を右に向けます。首を左に向けます。3回ずつ
4くびをまよこにむけます首を右に傾け5秒間とめます。首を左に傾け5秒間とめます。3回ずつ
5かたのあげさげをします首を回します。5回ほど回したのち反対に回します。10回
6かたをまわします うしろに5かい肩の上げ下げをします。前後10回ずつ
7てをくんでじょうはんしんをねじります上半身をねじります。手を組むか肘置きを持って、左右5秒間ずつ。3セット
8おやゆびをにぎってぐーをつくります手を斜め上方にあげ、胸を張るように後ろまで引きます。手を後ろに組んで上げ下げ。10回
9ひじからさきのくっしんをします親指を中に握り肘から先の屈曲をします。片方は伸ばした状態で交わします。10回/15回

5.2 午後体操プログラム（下半身）

順番声かけ（ひらがな）内容（スタッフ用）回数
1あしをかるくたたくかまっさーじをします息を吐きながら前かがみになり両手で足の甲を触ります。3回
2あきれすけんをのばします右足を水平に上げて10秒間保ちます。左右行います。5回×2セット
3あしをちゅうにあげてください足首を回します。右回し左回し各5回ずつ。左右各5回
4つまさきあげとかかとあげつま先を10回上げ下げ。かかとを10回上げ下げ。かかと上げ5秒×5セット。各10回
5あるくのでうでをふってください腕を振って足を合わせます。ゆっくりで15歩歩きます。10〜15歩
6あしを5びょうかんはやくうごかしますバタ足を5秒間行います。2セット
7しんこきゅうをします深呼吸をします。3回

5.3 指先・手の運動

  • 指を組んでゆっくりといろんな方向に回す
  • 指折りを10回行い、だんだん早くする
  • グーチョキパーを5回、逆（パーチョキグー）を5回
  • 合掌して指先を押し付け、指を順番に離して回す（反対回しも）
  • 手首を縦・横に動かし、回す
  • 左右違う動き（右手グー縦/左手パー前後）を入れ替える

5.4 口腔体操

順番声かけ内容回数
1あくびをします大きく口を開けてあくびの動作—
2くちのなかにくうきをいれてふくらまします口の中に空気をパンパンに入れて膨らます—
35びょうかんとめます空気を漏らさないよう口をチャックして5秒間止める2セット
4くちのなかのくうきをぬくようにすぼめます口をすぼめて空気を抜く。5秒間止める。2セット
5したでくちびるをなめます舌で唇をなめる—
6したではのまえがわをそうじします舌で歯の前側を掃除する—
7したではのうらがわをそうじします舌で歯の裏側を掃除する—
8したでほっぺをおしだします舌で頬を右左と交互に押し出す左右交互

発声練習（パタカラ体操）

ぱ ぱ ぱ ぱ ぱ（5回）
た た た た た（5回）
か か か か か（5回）
ら ら ら ら ら（5回）
ぱたから を続けて5回

あいうべ体操

「あいう」まで言って「べー」のところで舌を出します。5回繰り返します。

唾液腺マッサージ

唾液腺マッサージを行い、唾を飲み込む練習をします（2回飲み込み）。

最後に3回深呼吸をして終了します。

第6章 緊急時対応・感染症対策・通院

6.1 感染症発生時の対応

感染症（コロナ等）発生時のフロー:
  1. A棟・B棟の隔離対応を行う
  2. 該当者は居室での食事対応に切り替える（弁当等）
  3. 消毒を徹底する
  4. 発症から14日間経過を目安に解放する
  5. 関係機関（保健所等）に報告する

6.2 担当医院一覧と通院対応

付録: 使用書（本マニュアルの使い方）

1. 本マニュアルの目的

本マニュアルは、デイサービスセンターやわらぎの全職員が、日常業務を安全かつ効率的に遂行するための手引きです。新人教育、業務の標準化、サービス品質の維持向上を目的としています。

2. 対象者

  • 新人職員: 配属時に通読し、業務の全体像を把握してください
  • 既存職員: 業務手順の確認・振り返りに活用してください
  • 生活相談員: 第2章の事務業務フローを中心に参照してください
  • 管理者: 定例会議やカンファレンスの際の参考資料としてください

3. 使い方

場面参照箇所
朝の準備で何をすればよいか分からない第1章 1.1 朝の準備
利用者の食事形態を確認したい第4章 4.1 + おやつ形態一覧表

帳票を印刷したい各帳票の「印刷 / PDF保存」ボタン

4. 帳票の印刷・PDF保存方法

  1. 印刷したい帳票セクションに移動する
  2. 「印刷 / PDF保存」ボタンをクリックする（または Ctrl+P / Cmd+P）
  3. プリンターの選択画面で「PDFとして保存」を選ぶとPDFファイルとして保存できます
  4. 用紙サイズは A4 を選択してください
  5. 印刷時はナビゲーションやボタン等のUI要素は自動的に非表示になります

5. マニュアルの更新ルール

  • 業務手順に変更があった場合は、速やかに本マニュアルを更新すること
  • 更新は生活相談員または管理者が行う
  • 更新日を表紙の「最終更新」に反映すること
  • 定例会議（毎月20日以降）で変更内容を全職員に共有すること

6. 問い合わせ先

本マニュアルの内容に不明点がある場合は、生活相談員または管理者に確認してください。
